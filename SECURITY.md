# Security Policy

## Supported Versions

Currently, only the latest release is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of **PROXMUX Manager** seriously. If you believe you have found a security vulnerability, please do NOT open a public issue. Instead, please report it via one of the following methods:

1. **Email**: (Optionally) [INSERT EMAIL ADDRESS]
2. **GitHub Private Reporting**: Use the "Report a vulnerability" button in the Security tab of the repository if available.

We will acknowledge your report within 48 hours and provide a timeline for a fix. Please follow responsible disclosure practices.

## Our Commitment
- We will not collect or transmit user credentials.
- All sensitive data remains encrypted in local storage.
- We minimize permissions to only what is strictly necessary.
